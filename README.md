# rabota-13
### Условие задачи (вариант 19)

Слова в строке разделены пробелами, знак препинания располагается сразу после буквы и за ним следует пробел.
Разработайте и протестируйте функцию, которая читает первое слово из строки ввода в массив и отбрасывает остаток строки. Она должна пропускать ведущие пробелы. Определите слово как последовательность символов без пробелов, табуляций или новых строк. Используйте getchar(), а не scanf.

Составьте программу, которая во введённой пользователем строке находит слова с тире.
